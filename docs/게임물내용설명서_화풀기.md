## 2. 게임 주요 줄거리(시나리오) 요약

화풀기는 일상에서 쌓인 스트레스와 분노를, 실제 사람을 해치지 않고 **게임 안에서 안전하게 해소**하도록 돕는 캐주얼 게임이다. 이용자는 자신을 화나게 한 대상(회사, 고객, 가족 등)을 상징하는 만화풍 캐릭터를 화면에서 터치해 타격하며 스트레스를 푼다. 타격을 진행할수록 분노 수치가 줄어들고, 게임 종료 시 "분노 해소율"과 점수가 집계된다. 폭력 그 자체가 아니라 **감정 해소와 기록·성취**가 목적인 힐링·오락 게임이다.

---

## 3. 게임 진행 흐름

1. **인트로** — 게임 시작 화면("오늘 쌓인 화, 날려버릴까요?")
2. **대상 선택** — 화가 난 대상 유형 선택: 회사 / 고객 / 배우자 / 가족 / 친구 / 연인 / 기타
3. **닉네임 입력** — 대상에게 붙일 별칭 입력(자유 입력)
4. **분노 수치 설정** — 현재 분노 정도를 0~100으로 설정
5. **게임(타격) 플레이** — 화면의 캐릭터를 터치/연타하여 타격, 분노 게이지 감소
6. **결과** — 타격 수, 분노 해소율(%), 획득 점수 표시, 메모 작성 가능
7. **부가 기능** — 홈 대시보드, 랭킹, 업적/배지, 연속 플레이(스트릭), 주간 리포트, 출석 포인트

---

## 4. 주요 캐릭터

- **타격 대상 아바타** : 이용자가 선택한 대상(회사/고객/가족 등)을 상징하는 **만화풍 캐릭터**. 특정 실존 인물이 아니며, 이용자가 입력한 별칭(닉네임)이 캐릭터 위에 표시된다.
- **마스코트/안내 캐릭터** : 게임 진행을 돕는 귀여운 일러스트 캐릭터(불꽃 캐릭터, 글러브 등).

※ 모든 캐릭터는 비실사 만화풍이며, 유혈·신체 훼손·잔인한 묘사가 없다.

---

## 5. 주요 시스템 및 아이템

- **아이템/아이템 조합 시스템** : **없음** (수집·조합·강화형 아이템 없음)
- **점수(포인트) 시스템** : 타격 결과에 따라 점수 산정(아래 7항)
- **콤보/연타** : 빠르게 연속 타격 시 "연타!" 등 콤보 연출 및 타격감 강화
- **업적·배지** : 누적 타격·세션·해소율·연속 플레이 등 조건 달성 시 배지 부여(14종)
- **랭킹** : 점수·해소율 기준 주간/월간/전체 리더보드
- **연속 플레이(스트릭)** : 연속 플레이 일수 집계
- **출석 포인트** : 1일 1회 출석 시 포인트 적립(토스 프로모션 연동) — **확률·뽑기 등 사행 요소 없음**

---

## 6. 게임 조작방법

- **터치(탭)** : 화면 중앙의 캐릭터를 손가락으로 **터치하면 1회 타격**된다. ("캐릭터를 터치해서 때려보세요")
- **연타** : 빠르게 반복 터치하면 콤보가 쌓이며 타격 연출이 강화된다.
- 별도의 방향키·조이스틱·Hotkey는 없으며, **단일 터치 입력만으로 진행**되는 간단한 조작이다.

---

## 7. 전투(타격) 장면 설명

- 이용자가 캐릭터를 터치하면 **타격 이펙트(별 모양, 섬광, 흔들림)**와 함께 캐릭터가 반응한다.
- 타격이 누적되면 분노 게이지가 줄고, 캐릭터 위 말풍선에 **상황 대사**가 표시된다.
- **유혈·상해·사망 묘사, 무기 사용, 잔인한 연출은 전혀 없으며**, 만화적이고 코믹한 타격 연출(별·효과음·콤보 텍스트)로만 표현된다.
- `[스크린샷 1~3 첨부: 대상 선택 / 타격 플레이 / 결과 화면]`

---

## 8. 점수 획득 방법

게임 종료 시 다음 공식으로 점수가 산정된다(서버에서 재검증):

```
점수 = 10 + 타격수 + (스킬샷수 × 4) + floor((시작 분노 − 종료 분노) / 2)
```

- 타격을 많이 할수록, 분노를 많이 해소할수록 점수가 높아진다.
- 산정된 점수는 랭킹·업적에 반영된다.
- **현금·재화로 환전되지 않으며, 베팅·확률 요소가 없다.**

---

## 9. 대사집(스크립트) 발췌

타격 진행 단계별로 캐릭터/이용자 시점의 코믹한 해소성 대사가 표시된다. 비속어·욕설은 사용하지 않으며, 다소 비꼬는 구어체 표현이 포함된다.

- 초반: "왜 그렇게 당당해?", "오늘은 그냥 안 넘겨.", "내 기분은 생각 안 해봤지?"
- 중반: "이제 내 기분이 좀 이해돼?", "슬슬 뜨끔하기 시작하지?"
- 후반: "아~ 십 년 묵은 체증이 다 내려가네!", "이 정도면 정신이 좀 번쩍 들지?"
- 종료: "휴, 덕분에 스트레스 다 풀렸네.", "꼴 좋다. 오늘은 여기까지."

`[전체 대사 목록은 별첨 또는 본 문서에 추가 가능]`

## 10. 등급분류 관련 항목 자체 평가

| 항목 | 수준 | 설명 |
|------|------|------|
| 선정성 | 없음 | 노출·성적 표현 없음 |
| 폭력성 | 약함 | 만화풍 캐릭터 터치 타격. 유혈·상해·무기·사망 묘사 없음 |
| 공포 | 없음 | 공포 연출 없음 |
| 약물 | 없음 | 약물 관련 표현 없음 |
| 범죄 | 없음 | 범죄 묘사·조장 없음 |
| 언어 | 약함 | 비속어·욕설 없음. 비꼬는 구어체 표현 일부 |
| 사행성 | 없음 | 확률형 아이템·베팅·환전·뽑기 없음. 출석 포인트는 정액 적립 |

> 위 자체 평가에 따라 청소년이용불가 요소가 없으며, `[전체이용가 / 12세이용가]`로 신청한다.

## 12. 게임 종료

- 타격 세션 종료 시 결과 화면(해소율·점수)으로 이동하며, 이용자는 홈/랭킹/리포트 등으로 자유롭게 이동한다.
- 별도의 게임 클리어/엔딩 조건은 없는 **세션 반복형** 캐주얼 게임이다.
